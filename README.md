1. check your ruby version by running this command in the terminal:
```
$ ruby -v
```
2. remove `Gemfile.lock`
3. open `Gemfile` and change line 4:
```
ruby '<your_ruby_version>'
```
4. run these commands:
```
$ bundle install
$ yarn install
```
5. don't forget to create the database
```
$ rails db:create
```
6. run migrations
```
$ rails db:migrate
```
7. run the server
```
rails server
```
